# 🐍 Python Projects

Beginner Python projects built from scratch while learning Python fundamentals.

---

## Projects

--- Guess the Number
A number guessing game where the computer picks a random number (1-100) and the player guesses until correct.

**Concepts:** `random` module, `while` loops, `if/elif`, type casting, f-strings

---Password Generator
Generates secure random passwords of any length using letters, digits, and symbols. Saves passwords to a file.

**Concepts:** OOP, `random` + `string` modules, File I/O, class methods

---- To-Do List
A CLI task manager with full CRUD operations. Tasks persist between sessions using JSON.

**Concepts:** OOP, JSON, File I/O, dictionaries, lists, `enumerate()`
